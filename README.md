# lab-riscv-asm

Получить общее представление о системе команд RISC-V путём решения несложной
прикладной задачи.

## Вариант задания

Вариант задания | Передача параметров
--------------- | -------------
Найти среднее арифметическое значений на диагонали в матрице. (N = 4, M = 4) | Ввод в консоль

## Результаты работы

* **process.S** - алгоритм на RISC-V ASM
* **test.c**  - код на Си для проверки ASM модуля
* **main.c** - программа на Си
* **result.dump** - дамп-файл программы на Си

## Сборка
Сборка примера на Си в полностью настроенном окружении выполняется с использованием GNU Make. Для этого, находясь в каталоге с `Makefile` необходимо выполнить команду:
```
make
```

Для запуска тестового кода в QEMU необходимо выполнить команду:
```
make run
``
